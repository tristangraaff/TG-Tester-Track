De kracht van mixins ligt in dat je argumenten kan toevoegen, en zo dus kleine aanpassingen in een class op de originele mixin kunt maken. Met extend kunt je makkelijk code uit een andere class overnemen. Dit is handiger voor simpele doeleinden. Met mixins kun je dit ook doen alleen wordt er dan in het resultaat in CSS veel code herhaald, wat tegen het DRY principe ingaat.
