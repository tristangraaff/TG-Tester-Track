Opdracht 1 en 2

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze speudo-elementen inhouden.

  • 1. ::after - ::after creeert een pseudo-element dat de last child van het geselecteerde element is. Je kan dus content na / achter de content van het geselecteerde element plaatsen. Het is inline by default.

  • 2. ::backdrop - met ::backdrop wordt er een box ten grootte van de viewport onder alle andere van de pagina in full-screen mode getoond en kan deze worden gestyled.

    I'm a dialog window

  • 3. ::before - ::before creeert een pseudo-element dat de first child van het geselecteerde element is. Je kan dus content voor de content van het geselecteerde element plaatsen. Het is inline by default.

  • 4. ::cue - ???

  • 5. ::cue-region - ???

  • 6. ::first-letter - met ::first-letter kan je de style van de eerste letter van een element met display: block aanpassen. Dit kan alleen wanneer er geen andere content voor deze eerste letter staat.

  • 7. ::first-line - met ::first line kan je de styles van de eerste regel van een element met display:block aanpassen.

  • 8. ::file-selector-button - met ::file-selector-button kan je de styles van de button van een input element met de type="file" aanpassen.

  • 9. ::grammar-error - met ::grammar-error wordt aangegeven dat een user agend een stuk tekst als grammaticaal incorrect heeft gemarkeerd. Dit pseudo-element is experimenteel en op dit moment niet compatible met de bekendste browsers.

  • 10. ::marker - met ::marker kan het bolletje van een element met display: list-item worden gestyled, zoals li en summary.

  • 11. ::part - "The ::part CSS pseudo-element represents any element within a shadow tree that has a matching part attribute." Hiervoor is kennis van de Shadow DOM en JavaScript nodig om een voorbeeld te kunnen geven.

  • 12. ::placeholder - met ::placeholder kan je de styles van de placeholder tekst in een input of textarea element aanpassen.

  • 13. ::selection - met ::selection kan je de styles van het met de muis geselecteerde gedeelte van de pagina aanpassen. Probeer het uit door deze tekst te selecteren.

  • 14. The ::slotted() CSS pseudo-element represents any element that has been placed into a slot inside an HTML template (see Using templates and slots for more information). This only works when used inside CSS placed within a shadow DOM. Note also that this selector won't select a text node placed into a slot; it only targets actual elements.

  • 15. ::spelling-error - met ::spelling-error wordt aangegeven dat een user agend een stuk tekst met een spellingsfout heeft gemarkeerd. Dit pseudo-element is experimenteel en op dit moment niet compatible met de bekendste browsers.

  • 16. ::target-text - met ::target-text kan de tekst waar naartoe wordt gescrolld gestyled worden als de browser scroll-to tekstfragmenten ondersteunt.
