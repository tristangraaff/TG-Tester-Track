Met het woord 'cascade' wordt aangegeven dat iets van boven naar beneden loopt, zoals een waterval. In CSS houdt dit in dat de styles van boven naar beneden worden gelezen, en wanneer hier er conflicterende styles worden gelezen, de onderste style wordt toegepast en de eerdere styles overschrijft.

Met inheritance wordt aangegeven welke CSS props worden overgenomen uit parent elements. Dit is vaak te bedenken door je 'common sense' te gebruiken. Borders worden niet overgenomen, aangezien ik niet een border om ieder child van het element waar ik een border op heb ingesteld wil hebben.
