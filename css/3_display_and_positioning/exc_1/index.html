Opdracht 1

  • position: static;

    Dit is de default position. Element wordt volgens de flow van het document geplaatst.

  • position: relative;

    Element wordt op normale wijze geplaatst, met top right bottom en left kan het worden aangepast ten opzichte van de originele positie, d.w.z. relatief

  • position: absolute;

    Element wordt uit de flow van document gehaald en relatief aan z'n closest positioned ancestor geplaatst. Het kan dus boven op andere elementen komen, omdat het de flow van het document negeert.

  • position: fixed;

    Element wordt uit de flow van docu gehaald en er wordt geen ruimte voor gecreerd. Het komt relatief ten opzichte van de viewport te staan.

  • position: sticky;

    Element wordt uit de flow van docu gehaald en er wordt geen ruimte voor gecreerd. Het komt relatief ten opzichte van de nearest scrolling ancestor te staan.

  • position: inherit;
  • position: initial;
  • position: revert;
  • position: unset;
