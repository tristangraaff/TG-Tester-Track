Block box: - begint op een nieuwe regel - box vult de beschikbare ruimte van de container op, vaak is dit 100% - width en height kunnen toegepast worden - met padding, margin en border kunnen andere elements weg worden geduwd

Inline box: - de box wordt niet getoond op een nieuwe regel - width en height kunnen niet worden toegepast - verticale padding, margins en border kunnen worden toegepast, maar duwen andere inline boxes niet opzij - horizontale padding, margins en border kunnen worden toegepast en duwen andere inline boxes wel opzij
